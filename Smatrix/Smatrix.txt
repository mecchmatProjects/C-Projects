1) Введення:

У цьому проекті реалізовані функції для маніпулювання розрідженими матрицями і векторами на мові програмування C++. Основна мета-забезпечити ефективні інструменти для введення, виведення, обчислення визначників, обчислення зворотних матриць та вирішення систем лінійних рівнянь.

2) Бібліотека:

Цей проект використовує стандартні бібліотеки C++:

<iostream>:
Використовується для відображення результатів на екрані (std::cout, std::cerr).

<map>:
Він використовується для представлення розріджених векторів і матриць як асоціативних масивів.

<cmath>:
Він використовується для математичних операцій, таких як обчислення визначників та інших використовуваних виразів.

<fstream>:
Він використовується для зчитування даних з файлу у функції введення.

<sstream>:
Він використовується для управління стрічкою за допомогою функцій введення.


3) опис всіх файлів:

Smatrix.cpp:

SVector: Структура представляє розріджений вектор за допомогою std::map<int, Dtype> elements.
SMatrix: Структура представляє розріджену матрицю за допомогою std::map<std::pair<int, int>, Dtype> elements та має конструктор для ініціалізації розмірів.
input(SVector& x, ...): Функція отримує розріджений вектор та введену стрічку або дані з файлу, ініціалізує SVector з введеними значеннями та повертає кількість елементів.
input(SMatrix& x, ...): Функція отримує розріджену матрицю та введену стрічку або дані з файлу, ініціалізує SMatrix з введеними значеннями та повертає кількість елементів.
print(const SVector& x, ...): Функція отримує розріджений вектор та виводить його елементи.
print(const SMatrix& x, ...): Функція отримує розріджену матрицю та виводить її елементи.
determinant_recursive(const SMatrix& x, ...): Рекурсивно обчислює визначник матриці.
det(const SMatrix& x): Обчислює визначник матриці.
cofactor(const SMatrix& x, ...): Обчислює алгебраїчний додаток для конкретного елемента матриці.
inverse(const SMatrix& x): Обчислює обернену матрицю.
solve(const SMatrix& A, const SVector& y): Розв'язує систему лінійних рівнянь Ax = y та повертає розріджений вектор.


Smatrix.h:

SVector: Структура представляє розріджений вектор.
SMatrix: Структура представляє розріджену матрицю та має конструктор для ініціалізації розмірів.


Smatrix_test.cpp:

Тестовий код для перевірки функцій у файлі Smatrix.cpp.
Створює та виводить розріджений вектор, розріджену матрицю, обчислює визначник, обчислює обернену матрицю та розв'язує систему лінійних рівнянь.
Формули:


У файлі Smatrix.cpp, функція determinant_recursive використовує рекурсивний підхід для обчислення визначника матриці. Для матриць розміром 1x1 та 2x2 визначник обчислюється просто. Для матриць розміром більше 2x2 використовується рекурсивний розклад за першим рядком.
